Things I Make
Metro OS

Metro OS is a Sratch project made in TurboWarp (a Scratch mod).

The OS is a combo of Windows 8, iOS, Android 5 and old webOS.

The entire theme of 10Ui (the ui for Metro OS) is that it looks

like Windows 10 and Windows 8's Metro Ui.



Oxygen OS

Oxygen OS 5.0 is a Sratch project made in TurboWarp again.

It's a mix of macOS, Android 15, Windows 11 and iOS.
